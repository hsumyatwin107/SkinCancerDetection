# React Frontend (Vite)

## Run

```bash
cd app/frontend-react
npm install
npm run dev
```

Open: `http://127.0.0.1:5173`

Make sure FastAPI backend is running at `http://127.0.0.1:8000`.

## Features

- Image upload with preview
- Optional camera capture
- Predict request to `/predict` backend API
- Prediction label + confidence %
- Loading spinner while waiting for model response
